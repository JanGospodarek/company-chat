# miau

## How to run

### Requirements
- docker
- docker-compose

### Run

```bash
$ cd scripts
$ ./dev.sh
```

### Development

Edit the files locally and the changes will be reflected in the container. 


### Interact with the container

#### Logs
```bash
$ cd scripts
$ ./log.sh backend
```

#### Shell
```bash
$ cd scripts
$ ./shell.sh backend
```
